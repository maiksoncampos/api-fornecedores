
# api-fornecedores
Api para CRUD de fornecedores e endereços

## Tecnologias Implementadas:

- ASP.NET 6.0
- ASP.NET WebApi Core 
- Entity Framework Core 6.0
- .NET Core Native DI
- AutoMapper
- FluentValidator
- Swagger UI 

## Arquitetura:

- Separação de responsabilidades, SOLID e Clean Code
- Camada de Negócios
- Camada de Dados
- Notificações
- Repository
